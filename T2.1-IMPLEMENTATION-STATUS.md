# T2.1 用户认证模块后端 API - 实现状态报告

## 任务概述
实现用户登录、注册、JWT 令牌验证 API

## 技术栈
- Node.js + Express + TypeScript
- Prisma (SQLite 数据库)
- JWT (jsonwebtoken)
- bcrypt (密码加密)
- Zod (请求验证)
- Vitest (测试框架)

## 交付物状态

### ✅ 1. POST /api/auth/register - 用户注册接口
**位置**: `backend/src/controllers/auth.controller.ts`

**功能**:
- 接收邮箱、密码、姓名
- 验证邮箱格式和密码强度
- 检查邮箱是否已存在
- 使用 bcrypt 加密密码
- 创建新用户并返回用户信息

**验证规则**:
- 邮箱：有效邮箱格式
- 密码：8-128 字符，必须包含大写字母、小写字母和数字
- 姓名：可选，1-100 字符

**响应示例**:
```json
{
  "success": true,
  "data": {
    "user": {
      "id": "cm123abc...",
      "email": "john@example.com",
      "name": "John Doe",
      "role": "user",
      "createdAt": "2026-03-24T00:00:00.000Z"
    },
    "message": "User registered successfully"
  }
}
```

### ✅ 2. POST /api/auth/login - 用户登录接口
**位置**: `backend/src/controllers/auth.controller.ts`

**功能**:
- 验证邮箱和密码
- 使用 bcrypt 比较密码哈希
- 生成 JWT access token (15 分钟) 和 refresh token (7 天)
- 返回用户信息和令牌

**响应示例**:
```json
{
  "success": true,
  "data": {
    "user": {
      "id": "cm123abc...",
      "email": "john@example.com",
      "name": "John Doe",
      "role": "user"
    },
    "accessToken": "eyJhbGciOiJIUzI1NiIs...",
    "refreshToken": "eyJhbGciOiJIUzI1NiIs...",
    "expiresIn": 900,
    "message": "Login successful"
  }
}
```

### ✅ 3. GET /api/auth/verify - JWT 令牌验证接口
**位置**: `backend/src/controllers/auth.controller.ts`

**功能**:
- 验证 Authorization header 中的 JWT token
- 返回令牌有效性和用户信息
- 处理令牌过期和无效情况

**响应示例**:
```json
{
  "success": true,
  "data": {
    "valid": true,
    "user": {
      "userId": "cm123abc...",
      "email": "john@example.com",
      "name": "John Doe",
      "role": "user"
    },
    "expiresAt": "2026-03-24T01:00:00.000Z",
    "message": "Token is valid"
  }
}
```

### ✅ 4. authMiddleware.ts - 认证中间件
**位置**: `backend/src/middleware/auth.ts`

**中间件列表**:
- `authenticate`: 验证 JWT 令牌，附加用户信息到 request
- `requireAdmin`: 检查用户是否具有 admin 角色
- `optionalAuth`: 可选认证，有令牌则附加用户信息

**使用示例**:
```typescript
// 需要认证的路由
router.get('/me', authenticate, getCurrentUser);

// 需要管理员权限的路由
router.delete('/users/:id', authenticate, requireAdmin, deleteUser);

// 可选认证的路由
router.get('/public', optionalAuth, publicEndpoint);
```

### JWT 工具
**位置**: `backend/src/utils/jwt.ts`
- `generateAccessToken`: 生成 15 分钟有效期的访问令牌
- `generateRefreshToken`: 生成 7 天有效期的刷新令牌
- `verifyAccessToken`: 验证访问令牌
- `verifyRefreshToken`: 验证刷新令牌

**位置**: `backend/src/services/user.service.ts`

- `initializeAdmin`: 初始化管理员账户

3. **令牌过期**: Access token 15 分钟，Refresh token 7 天
4. **速率限制**: 认证接口 10 次/15 分钟
5. **输入验证**: Zod schema 验证
