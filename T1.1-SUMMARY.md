## 可用命令

### 根目录
```bash
pnpm install          # 安装所有依赖
pnpm dev             # 同时启动前后端开发服务器
pnpm build           # 构建前后端
pnpm lint            # 运行代码检查
pnpm format          # 格式化代码
pnpm test            # 运行测试
```

### 前端
```bash
cd frontend
pnpm dev             # 启动开发服务器 (http://localhost:8080)
pnpm build           # 生产构建
pnpm lint            # 代码检查
pnpm format          # 格式化代码
```

### 后端
```bash
cd backend
pnpm dev             # 启动开发服务器 (http://localhost:3000)
pnpm build           # 编译 TypeScript
pnpm start           # 启动生产服务器
pnpm lint            # 代码检查
pnpm format          # 格式化代码
pnpm test            # 运行测试
```

## 注意事项

1. **TailwindCSS v4**: 使用新的 CSS-first 配置方式 (`@import "tailwindcss"`)
2. **pnpm Workspace**: 使用 pnpm-workspace.yaml 管理多包项目

4. **Node.js 版本**: 要求 Node.js >= 20.0.0
