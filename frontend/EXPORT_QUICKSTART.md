# 数据导出功能 - 快速开始指南 (Quick Start Guide)

## 🚀 5 分钟快速上手

### 1. 访问导出页面

启动项目后，访问：
```
http://localhost:3000/export
```

或在侧边栏菜单点击 **"数据导出"**。

### 2. 配置导出任务

#### 第一步：选择数据类型
从下拉菜单中选择：
- Agent 数据
- 任务数据
- 日志数据
- 配置数据

#### 第二步：选择导出格式
- **CSV** - 适合 Excel 打开
- **JSON** - 适合程序处理
- **XLSX** - 适合 Excel 表格

#### 第三步：选择字段
- 点击字段前的复选框
- 或使用"全选"/"取消全选"按钮
- 已选字段数量实时显示

#### 第四步：设置过滤（可选）
根据数据类型设置过滤条件：
- 状态筛选
- 时间范围
- 类型筛选

### 3. 开始导出

点击 **"开始导出"** 按钮，系统会：
1. 验证配置（必须选择字段）
2. 创建导出任务
3. 自动跳转到进度页面

### 4. 查看进度

进度页面显示：
- ✅ 进度百分比
- 📊 记录数统计（总数/已处理）
- ⏱️ 剩余时间估算
- 🏷️ 当前步骤
- 🎯 状态图标

可执行操作：
- 取消导出（进行中时）
- 返回配置
- 刷新状态

### 5. 下载文件

导出完成后：
1. 显示 **"下载导出文件"** 按钮
2. 点击下载到本地
3. 可选择"导出其他数据"或"查看历史"

## 📋 常用场景示例

### 场景 1：导出所有任务数据为 Excel

1. 数据类型：**任务数据**
2. 导出格式：**XLSX**
3. 字段选择：**全选**
4. 过滤条件：**不设置**
5. 点击"开始导出"

### 场景 2：导出本周已完成的 Agent 数据

1. 数据类型：**Agent 数据**
2. 导出格式：**CSV**
3. 字段选择：ID、名称、状态、类型
4. 过滤条件：
   - 状态：活跃
   - 时间范围：本周
5. 点击"开始导出"

### 场景 3：导出错误日志用于分析

1. 数据类型：**日志数据**
2. 导出格式：**JSON**
3. 字段选择：ID、级别、消息、来源、时间
4. 过滤条件：
   - 日志级别：错误
   - 时间范围：今天
5. 点击"开始导出"

### 场景 4：查看和下载历史导出

1. 点击 **"查看历史"** 按钮
2. 使用搜索框查找特定记录
3. 使用筛选条件过滤：
   - 数据类型
   - 导出格式
   - 状态
   - 时间范围
4. 点击 **"下载"** 按钮获取文件
5. 可点击 **"删除"** 清理不需要的记录

## ⚠️ 注意事项

### 导出前
- ✅ 必须至少选择一个字段
- ✅ 建议选择必要的字段，减少文件大小
- ✅ 使用过滤条件减少数据量
- ⚠️ 大数据量导出可能需要较长时间

### 导出中
- ✅ 可以取消导出任务
- ✅ 进度每 2 秒自动更新
- ⚠️ 不要关闭浏览器页面
- ⚠️ 避免重复创建导出任务

### 导出后
- ✅ 及时下载文件（可能有保存期限）
- ✅ 清理不需要的历史记录
- ⚠️ 大文件下载需要耐心等待
- ⚠️ 注意文件存储位置

## 🔧 常见问题

### Q: 为什么导出很慢？
A: 可能原因：
- 数据量过大
- 网络速度慢
- 服务器负载高

建议：使用过滤条件减少数据量

### Q: 下载的文件打不开？
A: 检查：
- 文件格式是否正确
- 是否有对应的打开软件
- 文件是否完整下载

建议：尝试重新下载

### Q: 如何取消导出？
A: 在进度页面点击"取消导出"按钮

### Q: 历史记录在哪里？
A: 点击页面右上角"查看历史"按钮

### Q: 支持同时导出多个任务吗？
A: 建议等待当前导出完成后再创建新任务
