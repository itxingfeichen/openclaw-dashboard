# 备份管理功能实现文档

## 📋 概述

实现了完整的备份管理前端功能，支持手动备份创建、备份列表展示、备份恢复、备份删除和定时备份配置。

```
frontend/src/
├── pages/Backup/
│   ├── Backup.jsx          # 备份管理主页面
│   └── Backup.css          # 页面样式
├── components/
│   ├── BackupList.jsx      # 备份列表组件
│   ├── BackupSchedule.jsx  # 备份计划配置组件
│   └── RestoreDialog.jsx   # 恢复确认对话框
├── services/
│   └── backupService.js    # 备份 API 服务
└── tests/
    └── Backup.test.jsx     # 单元测试
```

## 🔌 API 接口

### backupService.js 提供的方法

```javascript
// 获取备份列表
fetchBackups(params: { page, pageSize, ... })

// 创建手动备份
createBackup(data: { type, description })

// 恢复备份
restoreBackup(backupId, options)

// 删除备份
deleteBackup(backupId)

// 获取备份计划
fetchBackupSchedule()

// 更新备份计划
updateBackupSchedule(schedule)

// 获取备份进度
fetchBackupProgress(backupId)
```

### API 基础 URL
```javascript
const API_BASE_URL = import.meta.env.VITE_API_URL || '/api'
```

### 模拟数据支持
所有 API 方法都包含模拟数据回退，当后端接口返回 404 时使用模拟数据，便于前端开发和演示。

## 🎯 使用方式

### 1. 访问备份管理页面
导航到 `/backups` 或在侧边栏点击"备份管理"

### 2. 创建手动备份
点击页面右上角的"创建备份"按钮

### 3. 恢复备份
在备份列表中找到状态为"完成"的备份，点击恢复图标，确认后输入"RESTORE"

### 4. 删除备份
点击备份列表中的删除图标，确认后删除

### 5. 配置自动备份
在右侧的"备份计划配置"卡片中：
1. 启用自动备份开关
2. 选择备份频率
3. 设置备份时间
4. 配置保留策略
5. 点击"保存"

## 📝 注意事项

1. **API 集成**: 当前使用模拟数据，需要后端实现 `/api/backups` 相关接口
2. **时间处理**: 使用 dayjs 进行时间格式化，确保已安装 dayjs 依赖
3. **权限控制**: 恢复和删除操作需要二次确认，防止误操作
4. **进度监控**: 使用轮询方式监控备份进度（2 秒间隔）
5. **错误处理**: 所有 API 调用都包含错误处理和用户提示
